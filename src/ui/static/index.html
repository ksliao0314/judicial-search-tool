搜尋說明

兩種模式

法院判決：各級法院（含最高、高行、地方法院等）的裁判書全文。

憲法解釋：大法官解釋（釋字第 1-813 號）＋ 新制憲法法庭裁判（111 年起憲判字）。

首頁切換器預設「法院判決」。憲法解釋走獨立資料庫（司法院大法官網），不混一般判決。

關鍵字語法

空格 = AND（同時包含）
例：「損害賠償 因果關係」→ 兩詞都出現

｜（全形豎線）或 OR = OR（擇一）
例：「僱傭｜雇用」「言論自由 OR 新聞自由」→ 任一出現即可

組合：「A B | C D」= (A AND B) OR (C AND D)

法院判決：自動展開

法條變體：「民法第179條」自動搜尋「第一百七十九條」「179條」等書寫形式。

法規簡稱：內建 71 組對照。「消保法第51條」自動擴展為「消費者保護法第51條」。

同義詞庫：已核准條目自動套用（如「僱傭 ↔ 雇傭」）。設定 → 同義詞庫可管理。

主文篩選：「主文含」欄位限縮裁判主文內容（例「上訴駁回」「撤銷原處分」）。選填。

憲法解釋：子字串比對

關鍵字直接在標題、爭點、理由書中做子字串匹配。不做法條變體展開、不用同義詞庫（資料量小、keyword 精準直擊）。

資料源：大法官網本機離線索引（868 筆、秒回）。不走司法院裁判書系統。

篩選與分析

搜尋完成後進入篩選頁面，可用法院層級 / 年度範圍 / 案由關鍵字限縮結果（憲法解釋模式無法院層級、年度 filter）。

理由篩選：勾選「僅保留理由含關鍵字」→ 先抓全文、比對理由段，只分析確實命中的判決，可大幅省 AI 成本。

AI 分析：輸入自然語言問題 → Claude 逐筆評分（0-10）＋ 摘錄原文 ＋ 立場／重點（60 字）＋ 全集綜合摘要。

體驗細節

跨任務快取：同一則判決若曾被其他任務抓過且同 parser 版本、30 天內，不重抓。

任務並行：最多 5 個任務同時跑（搜尋 / 抓全文 / AI 分析），互不阻塞。

中斷自動重試：伺服器重啟導致中斷的任務，開啟首頁時自動重試（若有 API key）。

閱讀器：AI 評價框可點擊跳回對應段落；A/D/J/K 鍵切換上下筆；ESC 關閉。
